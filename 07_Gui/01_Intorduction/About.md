Graphical User Interface (GUI) works in all desktop systems.

Packages in Java:
- `java.awt`
- `java.swing`*: this one is selected because it offers more functionality. 

GUI classes in `java.swing`: 
1. Container Classes: 
    - JPanel.
    - JFrame.
    - ...
2. Component Classes:
    - JLabel.
    - JButton.
    - JTextField.
    - ...
3. Helper Classes:
    - Graphics.
    - Font.
    - Color.
    - Dimension.
    - ...
